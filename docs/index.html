Three ways to train

Run a proven program, or do your own thing.

StrongLifts 5×5

Guided linear progression

The classic beginner program. Squat every session, alternating Workout A/B, +2.5 kg every time you hit all your reps, and an automatic deload when you stall.

  • Tap-cycle plate circles + per-side plate math
  • Warm-up ramps and empty-bar sets built in
  • Variants: Plus, Lite, Mini, Ultra, Ultra Max
Madcow 5×5

Weekly ramp, for intermediates

Where you go when linear stalls. Sets ramp to a top each day, a Friday PR set sets next week's weight, with heavy / light / intensity days and back-off volume.

  • Prescribed ramp checklist, top set called out
  • Progresses weekly, not every workout
  • One shared top per lift across the week
Freeform routine

Strong-style, your rules

For everything else. Set rows — weight × reps — with every set prefilled from last time. Timed holds and bodyweight sets, per-exercise rest. Nothing is prescribed; you drive.

  • Accessory, glute, mobility — any split you invent
  • Add exercises and sets on the fly
  • Remembers what you did last time
Program mode

Tap a plate. Set logged.

Each set is a plate you cycle with a tap — full reps, missed reps, skipped. The bar’s working weight, warm-up ramp, and per-side plate math are already worked out. You confirm; the app tracks.

  • Automatic +weight on success, deload after repeated fails
  • Warm-up ramps and empty-bar sets built in
  • Rest timer with a bell — counts through your set
The ramp

Ramped sets, done right.

Madcow and heavy days climb to a top set, then a PR set and a back-off — prescribed, checked off one tap at a time, with the top set called out. The name isn’t decoration: ramping is how the engine progresses you.

  • Heavy / light / intensity days, each with its own scheme
  • The Friday PR sets next week’s top automatically
  • Freeform routine mode too — every set prefilled from last time
Progress

See the line go up.

Estimated 1RM, top set, volume and total reps per lift — with a gold star the moment you beat an all-time best. Tap any lift for the full picture; the history calendar shows every session at a glance.

  • Personal records detected and marked, in Progress and History
  • Body-weight trend tracked alongside the lifts
  • Import your existing StrongLifts or Strong export via CSV
Self-host

Run your own in ~10 minutes.

Rampset deploys to Cloudflare’s free tier — Workers to serve the app, Cloudflare Access (also free) to put a login in front. No servers, no monthly bill for a household.

  1. Fork & clone the repo, then npm install.
  2. Add your lifters. Copy .env.example → .env.local and set names, units, and starting weights. No personal data lives in the repo.
  3. Deploy to Cloudflare Workers. One command via the OpenNext adapter. Free tier is plenty for a few users.
  4. Put Cloudflare Access in front. A free zero-trust login (email code or Google) so only your people get in — that’s your auth, no passwords to manage.
# get it running locally
git clone https://github.com/you/rampset
cd rampset && npm install
cp .env.example .env.local   # edit your lifters
npm run dev                  # → localhost:3000

# ship it to Cloudflare Workers
npm run deploy

✓ deployed to rampset.you.workers.dev
# then add Cloudflare Access (free) in the
# dashboard to gate it to your emails.
